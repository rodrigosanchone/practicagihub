Git Hub 

 

 

En la terminal  

Git init para iniciar 

Git status para revisar el estado de cada documento 

Crear el repositorio en git hub seguir los pasos de git hub  

En la consola poner git remote add origin “pegar la url del repositorio” 

Hacer cambios: git push -u origin master 

Agregar todos los docuementos git add –A 

Hacer un commit con  git commit – m “Escribir comentarios” 

 

Git diff muestra los cambios echos en los documentos 

Git log muestra los commit realizados
